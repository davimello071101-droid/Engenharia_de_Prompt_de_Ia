# 🚢 Batalha Naval

Projeto desenvolvido para faculdade utilizando HTML, CSS e JavaScript.

## 📖 Sobre o projeto

Este projeto consiste em um jogo de Batalha Naval para dois jogadores no mesmo computador.

O sistema possui:
- tabuleiro 10x10
- sistema de turnos
- ataques
- acertos e erros
- troca de jogador
- tela de vitória
- interface moderna e responsiva

Os navios são posicionados automaticamente de forma aleatória no início da partida.

---

# 🎮 Como jogar

1. Clique em **INICIAR JOGO**
2. O Jogador 1 realiza um ataque
3. Passe para o próximo jogador
4. O Jogador 2 realiza o ataque
5. O jogo continua até todos os navios serem destruídos
